# Operações e Comparações

### Exercício 1

**Descrição:**  
Verifique se o número 10 é igual à string '10' usando == e ===.

**Código de Exemplo / Solução:**

```javascript
let num = 10
let text = '10'
console.log(num == text)
console.log(num === text)
```
